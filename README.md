# House Price Prediction

A machine learning project that predicts the price of a house based on its features.

---

## What does it do?

You give it details about a house — like size, number of bedrooms, location — and it predicts what the house might cost.

---

## How it works

I used a dataset of house sales and trained a regression model to find the relationship between house features and prices.

---

## Tech used

- Python
- Scikit-learn (machine learning)
- Pandas (data processing)
- Jupyter Notebook

---

## How to run it

1. Open the notebook inside the `House_Price_Prediction_Project/` folder in Jupyter Notebook or Google Colab
2. Run all cells from top to bottom
